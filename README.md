# Simple Web Diary

This is an easy project for beginners.

You can use this repository to:
- Write daily diary notes
- Practice editing files
- Learn how a small web project works
- Build a habit of daily GitHub commits

How to use:
1. Open any file
2. Add a small diary entry
3. Save the file
4. Commit your change

Goal:
Write something small every day.

Notes:
Write your daily diary entries below.
